# Lab 02: Observability

## Goal

Use the repository health and telemetry checks to understand the observability scaffolding.

## Tasks

1. Run `bash scripts/check-health.sh`.
2. Run `bash scripts/check-telemetry.sh`.
3. Compare Compose-oriented and Kubernetes-oriented checks.

## Output

Summarize the limits of placeholder observability, including that these checks do not prove end-to-end RAN telemetry correctness, and propose one concrete extension.
