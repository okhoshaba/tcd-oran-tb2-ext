# Labs

This directory contains teaching-oriented lab guides for Tb2-ext.

- `lab-01` introduces the baseline telemetry scaffold.
- `lab-02` focuses on observability and health checks.
- `lab-03` introduces Kubernetes deployment.
- `lab-04` focuses on structured analysis and reporting.

These labs assume a software-only, research and education context. They do not validate RF/OTA behavior, SDR or COTS UE operation, full O-RAN RIC/xApp integration, or production-grade deployment.
