# Lab 03: Kubernetes Deployment

## Goal

Render and apply the local Kustomize overlay on a suitable local cluster.

## Tasks

1. Run `bash tests/validate-k8s.sh`.
2. Run `bash scripts/deploy-k8s-local.sh`.
3. Inspect namespace and deployment objects with `kubectl`.

## Output

Describe how the overlay model supports repeated student-group deployment and why it should not be presented as production-grade multi-tenant isolation.
