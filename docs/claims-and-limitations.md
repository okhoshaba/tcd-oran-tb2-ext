# Claims and Limitations

## Allowed Claims

- software-only RAN telemetry extension;
- Docker Compose and Kubernetes-oriented reproducibility scaffold;
- GitOps-ready structure;
- educational multi-group deployment model.
- repository skeleton or starter implementation for Tb2-ext.

## Not Allowed Claims

- full O-RAN-native RIC or xApp testbed;
- RF or OTA validation;
- SDR or COTS UE baseline;
- production-grade deployment;
- conformance-grade testing;
- complete Kubernetes scalability proof;
- benchmarked multi-cluster or multi-tenant guarantee.

## Usage Rule

When updating documentation, issues, pull requests or lab material, contributors should stay within the allowed claims and explicitly mark placeholders, templates and unvalidated elements. Validation scripts in this repository do not prove end-to-end RAN telemetry correctness.
