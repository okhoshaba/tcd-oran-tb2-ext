## Intended Relationship to the Academic Report

This repository is intended to serve as the implementation artefact accompanying an academic report on pragmatic orchestration for O-RAN-relevant testbeds between Docker Compose and Kubernetes.

The relationship is:

- the academic report explains the rationale, boundaries and evaluation framing;
- this repository captures the reproducible implementation scaffold for Tb2-ext;
- future integration work should continue to distinguish implemented artefacts from planned extensions.

No additional DOI or publication venue should be inferred from this file unless it is explicitly added to the repository later.
