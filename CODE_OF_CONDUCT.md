# Code of Conduct

## Our Standard

This repository supports collaborative academic and educational work. Contributors are expected to be respectful, precise and constructive in technical and non-technical discussion.

Examples of expected behavior:

- engaging respectfully across different levels of experience;
- grounding technical claims in evidence or clearly marked assumptions;
- giving actionable review feedback;
- acknowledging limitations and uncertainties.

Examples of unacceptable behavior:

- harassment or discrimination;
- personal attacks;
- deliberately misleading technical claims;
- disruptive or bad-faith collaboration.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests and related community spaces.

## Reporting

For conduct concerns, contact the repository maintainers through a private channel designated by the project owner. If no dedicated channel has yet been published, open a minimal private report path in repository administration before broadening collaboration.

## Enforcement

Maintainers may edit, moderate or remove contributions that do not meet this standard.
